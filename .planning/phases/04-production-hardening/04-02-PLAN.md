| Boundary | Description |
|----------|-------------|
| Railway dashboard → processo | Env vars chegam via `process.env` no boot; WEBHOOK_SECRET e DIGISAC_API_TOKEN são segredos de alto valor |
| Internet → Railway edge → container | TLS terminado no edge do Railway; container recebe HTTP em 0.0.0.0:$PORT |
| Container → Volume persistente | /data é filesystem privado do serviço; não exposto externamente |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07 | Information Disclosure | .env.example versionado | mitigate | Acceptance criteria explicitamente verifica que nenhum token real (`sk-proj-`, padrão de chave) está presente — apenas placeholders com comentários. |
| T-04-08 | Tampering | railway.json versionado | accept | Arquivo só contém configuração de deploy (paths e comandos), não segredos. Alterações requerem PR — fluxo normal de code review. |
| T-04-09 | Information Disclosure | Railway dashboard env vars | mitigate | Env vars sensíveis (WEBHOOK_SECRET, DIGISAC_API_TOKEN, OPENAI_API_KEY) gerenciadas pelo dashboard do Railway (não commitadas). Access ao dashboard é responsabilidade do owner do projeto Railway. |
| T-04-10 | Denial of Service | Webhook endpoint exposto publicamente | mitigate | Rate-limit existente (WBHK-05: 60 req/min por IP) já implementado na Phase 1. Helmet + token validation (WBHK-02) permanecem ativos. |
| T-04-11 | Spoofing | Webhook sem HTTPS | mitigate | Railway força HTTPS em domínios gerados; URL HTTP não é emitida. Constraint do projeto já exige HTTPS (CLAUDE.md). |
| T-04-12 | Elevation of Privilege | Volume /data acessível apenas ao processo | accept | Filesystem do container é isolado por padrão no Railway. paused.json é read/write apenas pelo processo Node que o criou. |
| T-04-13 | Denial of Service | Health check endpoint chamado em alta frequência | accept | /health é O(1) sem I/O; Railway poller usa intervalo padrão de ~30s. Custo desprezível. |
